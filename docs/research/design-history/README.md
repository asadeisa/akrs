# Design History

Frozen analysis reports produced while designing AKRS v1. They explain *why* the
framework is shaped the way it is — the Kernel, the STATE/close-out lifecycle, the
canonical `AGENTS.md` adapter, and the applicability scale.

These are **historical**. They are not maintained, not authoritative, and not
required to use AKRS. For current doctrine, read
[`../../framework/`](../../framework/).
